# Modul Magento: AMRE Arcade (Eckohaus Ltd)

Struktur pembangunan teknis yang dirancang untuk memproses persamaan berbasis rumus di dalam lingkungan Magento 2. Modul ini bertindak sebagai mesin pemrosesan sisi server (*server-side*), yang memfasilitasi integrasi kalkulasi berbasis Fortran ke dalam pengembangan aplikasi berbasis web dan perutean sisi server.

## ⚙️ Fungsionalitas Inti

* **Motor Kalkulasi:** Mengintegrasikan pemrosesan berbasis Fortran ke dalam infrastruktur sisi server, yang memungkinkan eksekusi persamaan berbasis rumus yang kompleks untuk pengembangan aplikasi.
* **Perutean Sisi Server:** Menyediakan kerangka kerja struktural untuk menilai dan merutekan tugas komputasi *high-gain* dalam arsitektur Magento.
* **Visibilitas Kalkulasi:** Demonstrasi langsung dan pengujian motor kalkulasi Fortran tersedia melalui [arcade.kupu-home.com/fortran-calculation-motor](https://arcade.kupu-home.com/fortran-calculation-motor).
* **Integrasi Frontend:** Menyebarkan lingkungan kanvas target yang dipetakan ke domain tertentu, dikompilasi dalam parameter `_extend.less` khusus.
